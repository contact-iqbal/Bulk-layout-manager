# Dokumentasi DLayout

Selamat datang di dokumentasi resmi DLayout. Aplikasi ini membantu Anda mengelola layout secara massal dengan mudah.

## Fitur Utama

- **Upload Massal**: Upload banyak gambar sekaligus.
- **Manajemen Data**: Simpan dan kelola data layout Anda.
- **Peta Lokasi**: Integrasi dengan peta untuk lokasi proyek.
- **Export PDF**: Ekspor hasil layout ke dalam format PDF siap cetak.
